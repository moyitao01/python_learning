# Python 学习练习项目

这是一个 Python 编程学习的练习项目，包含基础语法练习和实际应用案例。

## 项目结构

```
.
├── python_practice/          # Python 基础语法练习
│   ├── python1.py           # 变量和数据类型
│   ├── demo2.py - demo4.py  # 演示程序
│   ├── python1.py - python8.py  # 基础练习
│   ├── Python_tuple.py      # 元组操作
│   ├── Python_tuple2.py     # 元组进阶
│   ├── Python_data.py       # 数据结构
│   ├── python_def.py - python_def4.py  # 函数定义
│   ├── Python_def5.py       # 函数进阶
│   ├── python_open.py       # 文件操作
│   ├── python_obor.py       # 其他操作
│   ├── Pythin_set.py        # 集合操作
│   └── Python_dict.py       # 字典操作
│
└── two_text/               # 实际应用练习
    ├── test1.py - test11.py     # 基础测试
    ├── test2.py - test10.py     # 进阶测试
    ├── test_文件操作1.py        # 文件处理练习
    ├── test_文件操作2.py
    ├── test_文件操作1-2.py
    ├── test_文件操作2-1.py
    ├── test_turtle1.py          # Turtle 图形绘制
    ├── test_网页处理1.py        # 网页数据处理
    └── test_网页处理1-1.py
```

## 主要内容

### 1. Python 基础语法
- **变量和数据类型**：字符串、数字、布尔值等基本类型的使用
- **元组操作**：元组的创建、索引、切片、统计等操作
- **集合和字典**：Python 中重要的数据结构
- **函数定义**：从基础到进阶的函数编写练习
- **文件操作**：文件的读写和处理

### 2. 实际应用案例
- **文件处理**：从文本文件中提取和处理数据
- **网页处理**：从网页内容中提取图片 URL 等信息的处理
- **Turtle 绘图**：使用 Python Turtle 模块进行图形绘制
- **数据处理**：学生信息、成绩等数据的提取和格式化

## 使用说明

1. **环境要求**：
   - Python 3.x
   - 标准库即可运行，无需额外安装依赖

2. **运行方法**：
   ```bash
   # 进入对应目录
   cd python_practice
   # 运行 Python 文件
   python python1.py
   ```

3. **学习路径建议**：
   - 首先学习 `python_practice` 中的基础语法
   - 然后练习 `two_text` 中的实际应用案例
   - 建议按文件编号顺序学习，循序渐进
